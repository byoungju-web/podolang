FAQ

자주 묻는 질문

정말 사진 3장이면 되나요?
네. 방 사진 1~3장을 올리면 Vision MCP가 뷰·침대 수·스타일·조명을 읽어내고, 그 태그를 근거로 제목·설명·가격·번역이 자동 생성됩니다. 사진이 많을수록 태깅 정확도는 올라갑니다.
번역은 몇 개 언어까지 되나요?
기본 30개 언어를 병렬로 번역합니다. 영어·일본어·태국어·프랑스어·스페인어·독일어·중국어·베트남어 등, 각 시장에 맞는 해시태그(#tokyostay, #bangkokstay)까지 자동으로 붙습니다.
가격은 어떻게 정해지나요?
PriceLabs MCP가 나라별 성수기 계수를 적용합니다. 예를 들어 도쿄는 8월 +40%, 방콕은 12월 +60%처럼 시장별 피크를 자동 반영해 고정가로 인한 손해를 막습니다.
서버를 직접 운영해야 하나요?
아니요. 전체 파이프라인이 Cloudflare Workers + R2 + D1 + Queue 위에서 돕니다. npx wrangler deploy 한 줄이면 배포되고, 이후 스케일링·서버 관리는 신경 쓸 필요가 없습니다.
비용 구조가 어떻게 되나요?
월 $49 구독 + 예약 1% 수수료입니다. 방 개수 제한이 없어, 운영대행 20%와 비교하면 예약이 늘수록 절감 폭이 커집니다.
